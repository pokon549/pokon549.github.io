Django 新的摩登后台 Jet Admin

2019-08-10

  1. 1. 特点
    1. 1.1. 高度可自定义的 Dashboard
    2. 1.2. 漂亮
    3. 1.3. 测试项目也能用
  2. 2. 隐私

众所周知才怪，Django 的后台并不是那么好用。而且界面实在是过于丑陋，根本没办法好好地管理后台。
怎么办呢？pokon548 最近就找到一个特别好用的在线后台管理程序 —— Jet Admin。它不止能为在线项目提供服务，就连我们在本地上运行的测试项目（http://127.0.0.1　的这种），Jet Admin 也能提供后台管理服务！是不是很神奇。

特点

pokon548 光说好用当然不够，那就边上图边介绍好了。

高度可自定义的 Dashboard

哪些数据很关键要时刻关注？哪些数据要转换成图表？Jet Admin 都能帮你做。

比如 pokon548 自己有一个问卷调查的联手项目（Django 官方的新手教程）。通过选用 List，可以快速了解我有哪些 Questions，和哪些 Questions 有哪些 Votes…

漂亮

至于怎么个漂亮法，看看官方的介绍图就知道了。

测试项目也能用

对于 pokon548 这种正在学 Django 的人来说，runserver 自然是家常便饭的事。既然连本地 runserver Jet Admin 都能用，那有什么理由不试试这个有意思的在线后台呢？

隐私

Jet Admin 声称自己的服务器是不能主动碰不到部署端的数据的。必须借助在项目里面的 Jet Bridge 才能对数据进行操作。而且 Jet Bridge 是开源的。
所以，数据隐私问题应该不用太过于担心（况且我就是学习 Django，把数据拿走了也没什么意义）。
